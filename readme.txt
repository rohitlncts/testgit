This app is built to consolidate open issues in any git repository
How to access: 
1. Open the URL https://githubdashboard1.netlify.com/
2. Provide github repository URL in input box and submit
3. Click on any category label (number), it will populate the table with open issues


Feature: 
1. can list open issues and categorise based on open duration
2. Upon clicking on any category, will list all issues of that category
3. Upon clicking on title of the issue, will open the issue in next tab
